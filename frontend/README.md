# HouseHunt Frontend (Vite + React)

Quick start

1. Open a terminal inside `c:\intern\frontend`
2. Install dependencies:

```bash
npm install
```

3. Run dev server:

```bash
npm run dev
```

Notes
- The frontend expects the backend API at `http://localhost:5000/api` by default. To change, set `VITE_API_URL` in your environment when running Vite.
